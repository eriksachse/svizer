# Svizer 📐

A component that creates a container which is resizeable.

Install:

```bash
npm i svizer
```

Usage:

```html
<script>
  import Svizer from "svizer";
</script>

<Svizer
  left={100}
  top={100}
  width={200}
  height={100}
  background={"#f2f2f2"}
  showhandle={true}
  handlecolor={"brown"}
  showcorner={true}
  handlewidth={3}>Test text
</Svizer>
```
